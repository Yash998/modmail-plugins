# Countdowns Plugin

[p] is your prefix.

add it by doing `[p]plugins add aniket091/modmail-plugins/countdown`

# commands

| Name           | Description                                              |
| ------------   | ----------------                                         |
| `[p]countdown create`     | Starts a countdown!                                  |
| `[p]countdown delete`    |  Delets a countdown!                     |
| `[p]countdown update`    |  Updates a countdown!                                    |
