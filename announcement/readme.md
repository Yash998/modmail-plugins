# Announcement Plugin

[p] is your prefix.

add it by doing `[p]plugins add aniket091/modmail-plugins/announcement`

# commands

| Name                      | Description                                                   |
| ------------              | ----------------                                              |
| `[p]announcement quick`   | An simple way of making announcements!                        |
| `[p]announcement start`   | Start an interactive session to create announcement!          |
