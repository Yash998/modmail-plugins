# Action Plugin

[p] is your prefix.

add it by doing [p]plugins add aniket091/modmail-plugins/action

# commands

| Name        | Description         |
| ------------| ----------------    |
| `[p]baka`   | Call a user baka!   |
| `[p]cuddle` | Cuddle a user!      |
| `[p]feed`   | Feed a user!        |
| `[p]hug`    | Hug a user!         |
| `[p]kiss`   |  Kiss a user!       |
| `[p]pat`    |  Pat a user!        |
| `[p]poke`   |   Poke a user!      |
| `[p]slap`   |   Slap a user!      |
| `[p]smug`   |  Smug at a user!    |
| `[p]tickle` |  Tickles a user!    |
