# Fun Plugin

[p] is your prefix.

add it by doing `[p]plugins add aniket091/modmail-plugins/fun`

| ------------   | ----------------                                         |
| `[p]8ball`     | Ask 8 ball a question!                                   |
| `[p]choose`    | Choose between multiple options!                         |
